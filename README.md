# 프론트엔드 TDD 학습용 Repo

## 사용하는 기술

- React
- TypeScript
- Vite
- Vitest
- Pnpm

## 파일 위치

### 테스트 파일

`src/component/Counter.test.tsx`

### 컴포넌트 파일

`src/component/Counter.tsx`

## 테스트 방법

- 이미 작성되어 있는 `Counter.test.tsx`를 통과할 수 있도록 코드를 작성해주세요.
- 현재 존재하는 Counter 컴포넌트는 예시입니다.

## 실행 방법

```bash
 npm install
 npm test
 // 혹은
 npm test src/components/Counter.test.tsx -- --run
```
